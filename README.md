分享与交流一线红队知识与技术。星主参与近三十场攻防演练，均为红队。希望借助知识星球社群认识更多的人以及学习更多的知识与技术，与同样热衷网络安全红队技术的人共同进步，提高自身技术水平的同时为网络安全行业贡献绵薄之力。

1.小额付费方式（星球创建第1个月88，第2个月开始调至188，介意勿加）
